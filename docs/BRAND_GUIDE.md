# Titan X5-B GPU — Brand Guide

This guide establishes the visual identity and brand rules for the Titan X5-B GPU IP commercialization.

## 🎨 Color Palette

We use a modern, tech-forward color scheme emphasizing our open-source roots mixed with enterprise readiness.

- **Primary Green:** `#00C853` (Material Green 600) — *Use for main CTAs, highlights, and the GPU die accent.*
- **Accent Emerald:** `#10B981` (Tailwind Emerald 500) — *Use for gradients and hover states.*
- **Dark Slate:** `#1F2937` (Tailwind Gray 800) — *Use for deep backgrounds, text, and primary structure.*
- **Off-White:** `#F9FAFB` (Tailwind Gray 50) — *Use for light backgrounds.*

## ✍️ Typography

- **Primary Font (Sans):** `Inter` or `Roboto`. Use for all UI elements, headings, and body copy.
- **Monospace Font (Code):** `JetBrains Mono` or `Fira Code`. Use for all code snippets, specs, and Verilog syntax.

## 🖼️ Logo Usage

The Titan X5-B logo is a stylized GPU die with the brand name.
- **File:** `assets/titan_x5b_logo.svg`
- **Favicon:** `assets/favicon.ico` (Note: Currently an SVG payload. For production, convert to a true `.ico` using an online tool like favicon.io).

### 🚫 The "Don'ts"
- **Do not** stretch or distort the logo aspect ratio.
- **Do not** change the core colors (e.g., no red or blue recoloring).
- **Do not** add drop shadows or bevel effects to the logo.
- **Do not** place the logo on a visually cluttered background where it loses contrast.
