Technical Specifications

Logic & Compute

  • Gate Count: 3,030,603 Logic Cells (Yosys)
  • Streaming Multiprocessors: 4
  • SIMT Width: 32 Threads per SM
  • Tensor Cores: 256 PEs (16x16 Array) FP16

Memory & Interconnect

  • Memory Bus: 512-bit (GDDR7 PAM3 simulation ready)
  • Crossbar: AXI4 8-Master / 4-Slave
  • L1 Cache: Per SM, 4-Way Set Associative
  • Display: Integrated VGA Display Engine
